# ArabEduBot 🎓

منصة تعليمية متكاملة عبر تيليجرام لمساعدة الطلاب العرب في العثور على الجامعات والتخصصات المناسبة.

## المميزات
- 🔍 بحث متقدم عن الجامعات حسب الدولة.
- 🎓 عرض تفاصيل الكليات والأقسام والرسوم الدراسية.
- 🏢 معلومات عن السكن الجامعي وتكاليف المعيشة.
- 🌐 واجهة API برمجية كاملة باستخدام FastAPI.
- 🛠 لوحة تحكم إدارية لإدارة البيانات بسهولة.
- 🐳 دعم كامل لـ Docker و Docker Compose.

## هيكل المشروع
- `app/bot`: كود بوت التيليجرام.
- `app/api`: كود الـ API والـ Schemas.
- `app/models`: نماذج قاعدة البيانات.
- `app/admin`: إعدادات لوحة التحكم.

## طريقة التشغيل

### باستخدام Docker (الطريقة الموصى بها)
1. قم بتعديل ملف `.env` وإضافة توكن البوت الخاص بك.
2. قم بتشغيل الأمر:
   ```bash
   docker-compose up --build
   ```

### التشغيل اليدوي
1. تثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
2. إعداد قاعدة البيانات وتغذيتها بالبيانات:
   ```bash
   python -m app.database.seeder
   ```
3. تشغيل الـ API:
   ```bash
   uvicorn app.main:app --reload
   ```
4. تشغيل البوت:
   ```bash
   python -m app.bot.bot
   ```

## لوحة التحكم
يمكنك الوصول للوحة التحكم عبر الرابط: `http://localhost:8000/admin`
